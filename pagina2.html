Los Fundamentos

HTML (Lenguaje de Marcado de Hipertexto)

Es la estructura de la página. Define dónde van los títulos, párrafos, imágenes y enlaces.

CSS (Hojas de Estilo en Cascada)

Es la capa visual. Se encarga de los colores, tipos de letra, tamaños y la disposición de los elementos.

JavaScript

Es el lenguaje que da interactividad a la página (animaciones, formularios inteligentes, mapas, etc.).

Herramientas y Frameworks

Frameworks

Son conjuntos de herramientas y código ya escrito que ayudan a los programadores a trabajar más rápido y de forma organizada.

Angular / React / Vue.js

Son los tres frameworks (o librerías) de JavaScript más famosos para crear interfaces de usuario modernas.

Bootstrap

El framework de CSS que estás usando actualmente. Ayuda a diseñar sitios web rápidamente con componentes listos.

Tailwind CSS / Foundation:

Otras alternativas a Bootstrap para estilizar sitios web de forma eficiente.

Lógica y Datos (Backend)

Desarrollo Frontend

Todo lo que el usuario ve en el navegador (HTML, CSS, JS).

Desarrollo Backend

Lo que pasa "detrás de escena" en el servidor (procesar datos, seguridad, bases de datos).

Python / PHP / Node.js / .NET

Lenguajes y entornos utilizados para programar la lógica del servidor (Backend).

Bases de Datos SQL

Organizan la información en tablas (como Excel), ideales para datos muy estructurados.

Bases de Datos NoSQL

Guardan la información de forma más flexible, útiles para grandes volúmenes de datos variados.

Conectividad y Publicación

APIs

Puentes que permiten que dos aplicaciones se comuniquen entre sí (por ejemplo, mostrar el clima de Google en tu propia web).

Servidores Web

Computadoras potentes que almacenan los archivos de tu web y los "sirven" a los usuarios cuando entran a tu sitio.

Hosting (Alojamiento)

El servicio de alquiler de un espacio en un servidor para que tu web esté en internet.

Dominio

Es la dirección única de una web.

CMS (Sistemas de Gestión de Contenido)

Software como WordPress que permite crear webs sin escribir todo el código desde cero.

SEO

Estrategias para que tu página aparezca en los primeros resultados de búsqueda en Google.

CDN

Red de servidores repartidos por el mundo para que tu página cargue rápido, sin importar desde dónde se conecte el usuario.
